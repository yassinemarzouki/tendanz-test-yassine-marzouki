## Hints for Implementation

### Pricing Calculation
- Use `BigDecimal` for all monetary calculations (never use `double`)
- Always round to 2 decimal places with `RoundingMode.HALF_UP`
- The formula is straightforward: `base × age_factor × zone_coefficient`

### Exception Handling
- Be specific about which exception to throw
- Include meaningful error messages
- Always log exceptions with appropriate level

### Testing
- Use the setUp() method to create test data
- Test both success and failure cases
- Verify state changes (database writes)

### Code Quality
- Follow Spring Boot best practices
- Use dependency injection
- Keep methods focused and small
- Write meaningful variable names
- Add JavaDoc comments for public methods
